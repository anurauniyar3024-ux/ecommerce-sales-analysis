# 🛒 E-Commerce Sales Analysis

## Project Overview

This project analyzes the Brazilian Olist E-commerce dataset using Python.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

- Data Loading
- Data Cleaning
- Data Merging
- Exploratory Data Analysis (EDA)
- Business Insights
- Data Visualization

## Key Insights

- Total Revenue
- Total Orders
- Customer Analysis
- Monthly Sales Trend
- Top Product Categories
- Payment Method Analysis
- Customer Review Analysis
- Average Delivery Time
